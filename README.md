# Modeling collisions using Python and R 
---
This project explores computational physics for collision in 1-dimension 
and 2-dimensions.
---
## Physics
---
$$F_k=\mu_s\times N$$
$$N=m\times g$$
$$\triangle P=P_f-P_o\newline P=\sum_{i=1}^n m_iv_i$$
$$\sum_{i=1}^n m_i v_{i, o}=\sum_{i=1}^n m_i v_{i, f}$$
